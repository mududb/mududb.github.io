# MuduDB 文档

::::{div} mududb-hero
MuduDB 是一个面向过程、数据本地执行的数据库系统。它将应用程序过程以 WebAssembly 组件的形式部署在数据库运行时内部，从而使事务、调度、数据访问和业务逻辑保持在同一受控边界内。
::::
